Timestamp Microservice

Try it out:

API Response:

 
        

        


        

            This API allows you to get the Unix timestamp (in milliseconds) and the UTC date string for a given date.
        

        
Example Usage (Direct API links):

        
Pass a date string or a Unix timestamp (in milliseconds) to the /api/:date? endpoint.

        
    
            
  • [project url]/api/2015-12-25
    • 
            
  • [project url]/api/1451001600000
    • 
            
  • [project url]/api/ (for the current time)
    • 
        


        
Example Output:

        

            For a valid date: {"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
        

        

            If the input date string is invalid: {"error":"Invalid Date"}
