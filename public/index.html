Timestamp Microservice

This API endpoint converts date strings or Unix timestamps to a JSON object containing both the Unix timestamp (in milliseconds) and the UTC date string.

Usage Examples:

  • /api/2015-12-25
  • /api/1451001600000 (Unix timestamp in ms)
  • /api/ (current time)
  • /api/invalid-date-string (invalid date)
